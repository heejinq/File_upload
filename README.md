# Ajax 방식을 이용한 저장
- 파일 업로드시 원본파일과 섬네일용 파일이 uuid를 사용한 파일명으로 변경되며 업로드한 날짜에 맞는 폴더가 자동 생성되어 그곳에 파일이 저장된다.
  그리고 jsp화면에 섬네일 이미지가 보이게 된다.
